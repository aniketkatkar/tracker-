What is Task Tracker?


Task Tracker is the web based task tracking application. This tool helps to get visibility for the Managers on tracking the tasks assigned to the users.


Task Tracker helps by making the tasks assigned to the user, visible to the Manager. It contains tasks and sub tasks that the Managers assign which wont be updated in the map tool. This will help Managers check on all the Extra tasks performed by the user during the phase of the game and will help in evaluating the users performance.

Pages


User Panel: This Panel contains all the pages that are visible to the users to interact.

Login/Register: This is the Login and Register page which is used for authentication. User has to register in the application for the first time. This will be proceeded by a confirmation mail that the user gets in which an Activation link is present, which will activate your account. The user can then Login using the credentials they used while registering.

Dashboard: Dashboard contains various modules which the user can Add/Modify or Delete their Tasks. The User can Add tasks by clicking on 'Create a new task' button. It contains the Task name, Task description, Date created, Due date and completion date. The user can also add specific comments related to the Tasks. The user can also check on all the tasks they have entered. It also contains options to Update and Delete the created tasks.



Admin Panel: This Panel contains all the pages that are visible to the user to interact.

Login/Register: This is the Login and Register page which is used for authentication. Admin has to register in the application for the first time. This will be proceeded by a confirmation mail that the admin gets in which an Activation link is present, which will activate your account. The admin can then Login using the credentials they used while registering.

Dashboard: Admin Dashboard shows a list of Testers and the Tasks assigned to them. The Testers name is followed by the Task status of each tester. It is categorized as On Track, On Hold, Delayed, Completed and At Risk. The Admin can view the list of Tasks assigned to a specific Tester by clicking on the 'View Tasks' link.

View Task: The ‘View Task’ page is only available for the Admin panel. It shows the Tasks of each Tester with all the attributes like Task Status, Description and Dates.
